Overview

The Census Bureau collects data from people in the United States throughout the year. Federal, state, and local governments use the data to assess how constituents are represented and to allocate spending. The data is also made freely available to the public and has a wide range of use cases.1 In this post I parse, load, and validate data from the Census Bureau's American Community Survey (ACS) 2013 5-year Public Use Microdata Sample (PUMS) for Washington DC.

Brief process:

  • Start with a running Google Cloud VM instance and Jupyter Notebook server following "Running an Ipython Notebook on Google Compute Engine from Chrome".
  • Download (and decompress):2
    • 2013 5-year PUMS data dictionary: PUMS_Data_Dictionary_2009-2013.txt (<1 MB)
    • 2013 5-year PUMS person records for Washington DC: csv_pdc.zip (5 MB compressed, 30 MB decompressed)
    • 2013 5-year PUMS housing records for Washington DC: csv_hdc.zip (2 MB compressed, 13 MB decompressed)
    • 2015 5-year PUMS estimates for user verification: pums_estimates_9_13.csv (<1 MB)
  • Load the files:
    • Data dictionary TXT: dsdemos.census.parse_pumsdatadict TODO: link.
    • Person/housing and verification CSVs: pandas.read_csv link3
  • Verify calculations of estimates. Example:

To replicate the calculations in this row from the verification file:

  st  state  characteristic  pums_est_09_to_13  pums_se_09_to_13  pums_moe_09_to_13 
300  11  District of Columbia  Age 25-34  134,025  526  865 

With the verification file loaded as dfe, person records file loaded as dfp, and import numpy as np: 

# Select the verification estimates for Washington DC then for 'Age 25-34'.
# Select the corresponding data from the person records file.
tfmask_dc = dfe['state'] == 'District of Columbia'
dfe_dc = dfe.loc[tfmask]
char = 'Age 25-34'
tfmask_ref = dfe_dc['characteristic'] == char
tfmask_test = np.logical_and(25 <= dfp['AGEP'], dfp['AGEP'] <= 34)
# Calculate the estimate ('est', also a "characteristic"),
# a weighted sum, using sample weights 'PWGTP'.
ref_est = int(dfe_dc.loc[tfmask_ref, 'pums_est_09_to_13'].values[0].replace(',', ''))
test_est = dfp.loc[tfmask_test, 'PWGTP'].sum()
assert np.isclose(ref_est, test_est, rtol=0, atol=1)
# Calculate the direct standard error ('se') using replicate weights '[PWGTP[1-80]'.
# http://www2.census.gov/programs-surveys/acs/tech_docs/pums/accuracy/2009_2013AccuracyPUMS.pdf
# section 7, "Measuring Sampling Error"
# wts = ['PWGTP1', 'PWGTP2', ..., 'PWGTP80'] are the per-person
# 'replicate weights' for calculating 'direct standard error'
ref_se = dfe_dc.loc[tfmask_ref, 'pums_se_09_to_13'].values[0]
test_se = ((4/80)*((dfp.loc[tfmask_test, wts].sum() - test_est)**2).sum())**0.5
assert np.isclose(ref_se, test_se, rtol=0, atol=1)
# Calculate a margin of error ('moe') at the 90% confidence level (+/- 1.645 standard errors)
# http://www2.census.gov/programs-surveys/acs/tech_docs/pums/accuracy/2009_2013AccuracyPUMS.pdf
# section 7, "Measuring Sampling Error"
ref_moe = dfe_dc.loc[tfmask_ref, 'pums_moe_09_to_13'].values[0]
test_moe = 1.645*test_se
assert np.isclose(ref_moe, test_moe, rtol=0, atol=1)

Motivations

Why am I using the American Community Survey (ACS)?

  • The ACS is a relevant data set. A future step is to predict an individual's household income, which is among the subjects that the ACS survey addresses.
  • The ACS is a cultivated, thorough data set.5 The ACS has many quality controls to ensure that it is representative. The survey samples about 3 million addresses per year with a response rate often over 95%.
  • The ACS is a timeseries data set. The survey sends questionnaires throughout the year and releases data once per year. As a timeseries, the data can be used for forecasting analytics, which is future step.
  • I recognize that using ACS data can be problematic. Data from the Census Bureau has been used for harm,6 and current ACS terminology asks respondents to identify by terms such as "race".7 For this project, I take data from the Census Bureau at face value and I infer from it at face value. It's important to respect that these aren't simply data points; these are people.

Why am I using the ACS 5-year estimate?

As of Dec 2015, the ACS offers two windowing options for their data releases, 1-year estimates and 5-year estimates.8 Because the ACS 5-year estimates include data over a 5-year window, they have the largest sample size and thus the highest precision for modeling small populations. However, the sample size comes at the expense of currency. Forecasting the predictions from a 5-year window to be relevant to a specific year is a future step.

Why am I using Python?

This project can be done using Python, R, and/or other languages.4 I'm using Python since a future step is to make an extensible, optimized pipeline, which is a popular application for scikit-learn.

What about "big data"?

I'm staring with a data set small enough to be processed in memory (i.e. operated on in RAM), since many Python packages (e.g. the PyData stack) are most often used for in-memory operations on single machines. Within these packages, operations are often parallelized across the cores within the machine. For operations that exceed the available RAM on the machine (i.e. out-of-core computations), there's Dask for Python, and for operations that require a cluster of machines, there's Spark for Java, Scala, Python, and R. Scaling a pipeline to large enough data that requres a cluster is a future step.
